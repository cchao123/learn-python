## 💬 学习心得

### 最大的收获
- 发现 Python 比 JavaScript 更简洁
- 列表推导式非常强大
- 缩进代码更易读
- 类型检查更严格，避免很多错误

### 最大的挑战
- 理解 `range()` 函数的参数
- 复杂逻辑的条件判断
- 记住所有的语法细节

### 有趣的发现
- Python 没有 `switch`（Python 3.10+ 有 `match/case`）
- 负索引很方便
- `and/or/not` 比 `&&/||/!` 更直观
- 列表可以混合类型
